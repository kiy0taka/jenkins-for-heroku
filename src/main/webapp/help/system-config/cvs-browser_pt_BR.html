Especifica o caminho para a aplicação web de visualização do repositório CVS. Definir este valor permite que o Jenkins gere links das mudanças para estas páginas. Existem duas entradas diferentes para cada CVSROOT.

URL
A URL da página que mostra os detalhes sobre um arquivo particular no repositório.
URL de comparação
A URL da página que mostra detalhes sobre a comparação entre duas revisões.

Os seguintes caracteres de macro estão disponíveis.
%%P Nome do caminho do arquivo sob o CVS, como "foo/bar/Zot.java"
%%r Revisão da versão antiga (para URLs de comparação)
%%r Revisão da nova versão (para URLs de comparação)
