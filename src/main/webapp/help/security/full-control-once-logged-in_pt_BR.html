Neste modo, todo usuário logado obtém o controle total do Jenkins. O único usuário que não terá controle total é o usuário anônimo, que apenas consegue acesso a leitura.

Este modo é útil para forcar usuários a logar antes de fazer alguma ação, assim você pode manter registro de quem fez o quê. Esta configuração também pode ser usada numa instalação pública do Jenkins, onde você apenas permite usuários confiáveis a ter conta de usuário.
